# فزیکل AI ورک اسٹیشن (The Physical AI Workstation)

## ہارڈویئر کیوں اہمیت رکھتا ہے (Why Hardware Matters)

ویب ڈویلپمنٹ یا ڈیٹا سائنس کے برعکس، **فزیکل AI ڈویلپمنٹ ہارڈویئر پر بھاری (hardware-intensive)** ہے۔ آپ صرف نیورل نیٹ ورکس کو تربیت نہیں دے رہے—آپ فزکس کی نقالی کر رہے ہیں، 3D ماحول رینڈر کر رہے ہیں، اور ریئل ٹائم روبوٹ کنٹرولرز کو بیک وقت چلا رہے ہیں۔

غلط ہارڈویئر اس سفر کو سست سیمولیشنز اور میموری ختم ہونے والے کریشز کی مایوس کن دلدل میں بدل دے گا۔ **درست ہارڈویئر** ڈویلپمنٹ کو جادو کی طرح محسوس کرائے گا۔

یہ باب آپ کو فزیکل AI ڈویلپمنٹ کے لیے موزوں ورک اسٹیشن بنانے یا کنفیگر کرنے میں رہنمائی کرے گا۔

## اجزاء کی گہرائی میں (Component Deep Dive)

### 🎮 **GPU: فزیکل AI کا دل**

#### NVIDIA RTX کیوں؟

NVIDIA GPUs میں **CUDA cores** ہوتے ہیں جو ان کے لیے ضروری ہیں:
*   **Isaac Sim:** Omniverse پر بنایا گیا ہے، فزکس اور رینڈرنگ کے لیے CUDA کی ضرورت ہے
*   **PyTorch/TensorFlow:** ڈیپ لرننگ فریم ورکس ٹریننگ کے لیے CUDA استعمال کرتے ہیں
*   **OptiX ری ٹریسنگ:** فوٹو ریلسٹک سینسر سیمولیشن (کیمرے، LiDAR)

:::warning AMD GPUs سپورٹڈ نہیں ہیں
AMD Radeon GPUs (یہاں تک کہ RX 7900 XTX جیسے ہائی اینڈ بھی) **Isaac Sim نہیں چلا سکتے**۔ روبوٹکس ورک لوڈز کے لیے ROCm سپورٹ محدود اور غیر مستحکم ہے۔
:::

### 🧠 **CPU: فزکس انجن**

فزکس سیمولیشنز (Gazebo, Isaac Sim) ان کے لیے **CPU-bound** ہیں:
*   تصادم کا پتہ لگانا (Collision detection)
*   رگڈ باڈی ڈائنامکس (Rigid body dynamics)
*   رکاوٹ حل کرنا (جوڑ، موٹرز)

#### 8+ Cores کیوں اہم ہیں

جدید سمیلیٹرز فزکس کو CPU کورز پر متوازی چلاتے ہیں:

```python
# مثال: Gazebo بیک وقت 10 روبوٹس کی نقالی کر رہا ہے
# ہر روبوٹ کو فزکس کے حساب کتاب کے لیے 1-2 CPU تھریڈز کی ضرورت ہوتی ہے
# کل CPU کا استعمال: 10 روبوٹس × 2 تھریڈز = 20 تھریڈز
# 8-کور CPU (16 تھریڈز) پر: 50-60 FPS پر چلتا ہے
# 4-کور CPU (8 تھریڈز) پر: 15-20 FPS پر چلتا ہے ⚠️
```

#### تجویز کردہ CPUs

**Intel (13th/14th Gen):**
*   Intel i7-13700K (16 cores, 24 threads) – $350
*   Intel i9-13900K (24 cores, 32 threads) – $500

**AMD Ryzen (7000 Series):**
*   AMD Ryzen 9 7900X (12 cores, 24 threads) – $400
*   AMD Ryzen 9 7950X (16 cores, 32 threads) – $550

:::info بجٹ آپشن
اگر بجٹ تنگ ہے تو CPU پر GPU کو ترجیح دیں۔ RTX 4070 Ti + Ryzen 7 7700X (8-core) ایک RTX 4060 + Ryzen 9 7950X سے بہتر کارکردگی دکھائے گا۔
:::

### 💾 **RAM: ملٹی ٹاسکنگ بفر**

فزیکل AI ڈویلپمنٹ میں ایک ساتھ کئی بھاری ایپلی کیشنز چلانی پڑتی ہیں:

```bash
# عام ڈویلپمنٹ سیشن میموری کا استعمال:
ROS 2 Master Node:     2GB
Gazebo Simulator:      4GB
RViz (Visualization):  3GB
Isaac Sim:             8GB
VS Code + Extensions:  2GB
Docker Containers:     3GB
Chrome (Documentation):2GB
----------------------------
کل:                   24GB
```

**کم از کم:** 16GB (قابل استعمال لیکن تنگ)
**تجویز کردہ:** 32GB (آرام دہ)
**پرجوش:** 64GB (زیادہ تر کے لیے ضرورت سے زیادہ)

#### DDR4 بمقابلہ DDR5؟

| قسم | رفتار | قیمت کا فرق | سفارش |
| :--- | :--- | :--- | :--- |
| DDR4 | 3200-3600 MHz | بنیادی | ✅ زیادہ تر صارفین کے لیے کافی |
| DDR5 | 5200-6000 MHz | +30% | اختیاری؛ کم سے کم کارکردگی کا فائدہ |

**فیصلہ:** DDR4 ٹھیک ہے۔ بہتر GPU/CPU کے لیے پیسے بچائیں۔

### 💿 **اسٹوریج: تیز I/O اہمیت رکھتا ہے**

#### NVMe SSD کیوں؟

روبوٹکس سیمولیشنز **گیگا بائٹس اثاثوں** کو لوڈ کرتے ہیں:
*   Isaac Sim ماحول: 500MB - 5GB فی منظر
*   Docker امیجز (ROS 2 + انحصار): 10GB+
*   سیمولیشن لاگز (rosbags): 1GB فی گھنٹہ

**NVMe SSD بمقابلہ SATA SSD:**

| میٹرک | SATA SSD | NVMe SSD |
| :--- | :--- | :--- |
| پڑھنے کی رفتار | 550 MB/s | 3,500 MB/s |
| منظر لوڈ ہونے کا وقت | 15 سیکنڈ | 3 سیکنڈ |
| Docker امیج پل | 60 سیکنڈ | 12 سیکنڈ |

**کم از کم:** 256GB NVMe SSD
**تجویز کردہ:** 500GB NVMe SSD
**ڈیٹا ذخیرہ کرنے والے:** 1TB NVMe + 2TB HDD ڈیٹاسیٹس کے لیے

:::tip تنصیب کی حکمت عملی
1.  **NVMe SSD:** آپریٹنگ سسٹم، ROS 2، Isaac Sim
2.  **HDD/سیکنڈری SSD:** ڈیٹاسیٹس، rosbag لاگز، Docker والیومز
:::

### 💻 **آپریٹنگ سسٹم: Ubuntu 22.04 LTS (واحد انتخاب)**

#### Ubuntu کیوں؟

**ROS 2 Humble** (وہ ورژن جو ہم استعمال کرتے ہیں) سرکاری طور پر **Ubuntu 22.04 LTS** کو ہدف بناتا ہے:
*   `apt install` کے ذریعے پری بلٹ بائنری پیکجز دستیاب ہیں
*   روبوٹکس لائبریریوں کے ساتھ زیادہ سے زیادہ مطابقت
*   10+ سال کی کمیونٹی سپورٹ اور ٹیوٹوریلز

#### Windows یا macOS کیوں نہیں؟

| OS | ROS 2 سپورٹ | Isaac Sim سپورٹ | فیصلہ |
| :--- | :--- | :--- | :--- |
| **Ubuntu 22.04** | ✅ نیٹو، مکمل سپورٹ | ✅ نیٹو، بہترین کارکردگی | ⭐ **تجویز کردہ** |
| **Windows 11** | ⚠️ صرف WSL2، محدود | ⚠️ چلتا ہے، لیکن آہستہ | پروڈکشن کے لیے بچیں |
| **macOS** | ❌ سپورٹڈ نہیں | ❌ سپورٹڈ نہیں | ❌ **ناموافق** |

:::danger macOS ناموافقت (Incompatibility)
ایپل سلیکون (M1/M2/M3) میک **ROS 2 یا Isaac Sim کو مقامی طور پر نہیں چلا سکتے**۔ یہاں تک کہ ورچوئلائزیشن (UTM, Parallels) کے ساتھ بھی، GPU پاس تھرو سپورٹ نہیں ہے۔ **آپ کو NVIDIA GPU کے ساتھ پی سی کی ضرورت ہے۔**
:::

#### ڈوئل بوٹ بمقابلہ WSL2 بمقابلہ نیٹو

**آپشن 1: نیٹو Ubuntu (تجویز کردہ)**
*   ✅ مکمل ہارڈویئر رسائی، بہترین کارکردگی
*   ✅ کوئی ہائپر وائزر اوور ہیڈ نہیں
*   ❌ وقف شدہ پارٹیشن یا الگ ڈرائیو کی ضرورت ہے

**آپشن 2: ڈوئل بوٹ (Windows + Ubuntu)**
*   ✅ گیمنگ/دیگر سافٹ ویئر کے لیے ونڈوز رکھیں
*   ✅ Ubuntu میں ہونے پر مقامی لینکس کارکردگی
*   ⚠️ OS تبدیل کرنے کے لیے ریبوٹ کرنے کی ضرورت ہے

**آپشن 3: WSL2 (Windows Subsystem for Linux)**
*   ✅ ریبوٹ کی ضرورت نہیں، ونڈوز کے ساتھ لینکس چلائیں
*   ❌ 10-20% کارکردگی کا جرمانہ
*   ❌ GPU پاس تھرو مشکل ہو سکتا ہے
*   ❌ محدود USB ڈیوائس تک رسائی (حقیقی روبوٹس کے لیے اہم)

**ہماری سفارش:** Ubuntu 22.04 کو نیٹو طور پر انسٹال کریں یا ڈوئل بوٹ کریں۔ WSL2 سیکھنے کے لیے قابل قبول ہے، لیکن آپ رکاوٹوں کا سامنا کریں گے۔

## Ubuntu 22.04 انسٹالیشن گائیڈ

### مرحلہ 1: Ubuntu ڈاؤن لوڈ کریں

1.  دیکھیں [ubuntu.com/download/desktop](https://ubuntu.com/download/desktop)
2.  **Ubuntu 22.04.3 LTS** ڈاؤن لوڈ کریں
3.  SHA256 چیکسم کی تصدیق کریں

### مرحلہ 2: بوٹ ایبل USB بنائیں

**ونڈوز پر:**
1.  [Rufus](https://rufus.ie/) ڈاؤن لوڈ کریں
2.  Ubuntu ISO منتخب کریں
3.  "GPT" پارٹیشن اسکیم منتخب کریں
4.  "Start" پر کلک کریں

**macOS/Linux پر:**
```bash
# USB ڈیوائس تلاش کریں
lsblk

# ISO فلیش کریں (/dev/sdX کو اپنی USB ڈرائیو سے تبدیل کریں)
sudo dd if=ubuntu-22.04.3-desktop-amd64.iso of=/dev/sdX bs=4M status=progress
sudo sync
```

### مرحلہ 3: USB سے بوٹ کریں

1.  کمپیوٹر کو دوبارہ شروع کریں
2.  BIOS/UEFI میں داخل ہوں (عام طور پر F2, F12, یا Del)
3.  USB کو پہلے بوٹ ڈیوائس کے طور پر سیٹ کریں
4.  محفوظ کریں اور باہر نکلیں

### مرحلہ 4: Ubuntu انسٹال کریں

1.  "Install Ubuntu" منتخب کریں
2.  زبان اور کی بورڈ لے آؤٹ منتخب کریں
3.  **انسٹالیشن کی قسم:**
    *   **ڈسک مٹائیں اور انسٹال کریں:** موجودہ OS کو تبدیل کرتا ہے
    *   **کچھ اور (Something else):** ڈوئل بوٹ کے لیے (ایڈوانسڈ)
4.  یوزر اکاؤنٹ بنائیں
5.  انسٹال کریں (20-30 منٹ لگتے ہیں)
6.  ریبوٹ کریں

:::info ڈوئل بوٹ سیٹ اپ
اگر ونڈوز کے ساتھ ڈوئل بوٹنگ کر رہے ہیں تو، **"Install Ubuntu alongside Windows"** آپشن کا استعمال کریں۔ Ubuntu پارٹیشننگ کو خود بخود سنبھال لے گا۔
:::

### مرحلہ 5: NVIDIA ڈرائیورز انسٹال کریں

```bash
# پیکیج لسٹس کو اپ ڈیٹ کریں
sudo apt update

# NVIDIA ڈرائیور انسٹال کریں (تجویز کردہ)
sudo ubuntu-drivers autoinstall

# ریبوٹ
sudo reboot

# ڈرائیور کی تنصیب کی تصدیق کریں
nvidia-smi
# GPU کی معلومات اور ڈرائیور کا ورژن دکھانا چاہیے
```

## تصدیقی چیک لسٹ (Verification Checklist)

سیٹ اپ کے بعد، تصدیق کریں کہ آپ کا سسٹم تیار ہے:

```bash
# GPU چیک کریں
nvidia-smi
# متوقع: GPU کا نام، ڈرائیور ورژن 535+، CUDA 12.0+

# CPU کورز چیک کریں
nproc
# متوقع: 16+ (8-کور CPU کے لیے ہائپر تھریڈنگ کے ساتھ)

# RAM چیک کریں
free -h
# متوقع: 30+ GB دستیاب

# ڈسک اسپیس چیک کریں
df -h /
# متوقع: 200+ GB مفت

# Ubuntu ورژن چیک کریں
lsb_release -a
# متوقع: Ubuntu 22.04.3 LTS
```

:::tip ٹربل شوٹنگ
اگر `nvidia-smi` "NVIDIA-SMI has failed" دکھاتا ہے، تو آپ کے ڈرائیورز صحیح طریقے سے انسٹال نہیں ہوئے ہیں۔ دیکھیں [NVIDIA ڈرائیور ٹربل شوٹنگ گائیڈ](https://docs.nvidia.com/cuda/cuda-installation-guide-linux/)۔
:::

## لاگت کی اصلاح کی حکمت عملی (Cost Optimization Strategies)

### بجٹ: $1,000 سے کم

**کنفیگریشن:**
*   GPU: RTX 4060 Ti (8GB) – $400
*   CPU: AMD Ryzen 5 7600X – $200
*   RAM: 16GB DDR5 – $60
*   اسٹوریج: 500GB NVMe SSD – $40
*   مدر بورڈ: B650 – $150
*   PSU: 650W – $80
*   کیس + کولر: $100

**کل:** $1,030

**تجارت (Trade-offs):**
*   صرف سنگل روبوٹ سیمولیشنز
*   سیمولیشن کی پیچیدگی کو کم کرنے کی ضرورت پڑ سکتی ہے
*   سیکھنے کے لیے کافی، پروڈکشن کے لیے نہیں

### مڈ رینج: $1,500 - $2,000

**کنفیگریشن:**
*   GPU: RTX 4070 Ti (12GB) – $600
*   CPU: AMD Ryzen 9 7900X – $400
*   RAM: 32GB DDR5 – $120
*   اسٹوریج: 1TB NVMe SSD – $80
*   مدر بورڈ + PSU + کیس: $500

**کل:** $1,700

**صلاحیتیں:**
*   Isaac Sim میں متعدد روبوٹس
*   آرام دہ ٹریننگ ورک فلوز
*   اسٹارٹ اپس/فری لانسرز کے لیے پروڈکشن کے لیے تیار

### ہائی اینڈ: $3,000+

**کنفیگریشن:**
*   GPU: RTX 4090 (24GB) – $1,600
*   CPU: AMD Ryzen 9 7950X3D – $550

*   اسٹوریج: 2TB NVMe SSD – $150
*   پریمیم اجزاء: $500

**کل:** $3,040

**استعمال کے معاملات:**
*   تحقیقی لیبز
*   بڑے پیمانے پر ری انفورسمنٹ لرننگ
